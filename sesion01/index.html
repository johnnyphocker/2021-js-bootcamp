Cómo subir al repositorio

  • Crear carpeta.
  • Entrar a la carpeta.
  • Correr git init.
  • Crear algun documento, como por ejemplo el README.md
  • Salvar cambios.
  • Correr git add . .
  • Correr git commit -m 'First commit' como ejemplo.
  • Correr git branch -M main.
  • Correr git push origin main
  • Identificarte con usuario y contraseña.
